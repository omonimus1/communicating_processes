# Communicating processes

Client-server is a java application that allows a client to interact with a Server, 
by requesting the content of a file stored in the server (if this exists);

## How to run 
* Run first the server
* Then run the client

### From terminal
```
javac Server.java
// run Server
java Server

javac Client.java
java Client
```
